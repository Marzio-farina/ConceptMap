SceneManager

La classe SceneManager del namespace UnityEngine.SceneManagement è la principale interfaccia per la gestione delle scene in Unity. Questa classe fornisce metodi statici per caricare, scaricare e gestire le scene, consentendo ai giochi e alle applicazioni di passare tra diverse scene o livelli in modo dinamico.

Panoramica della Classe SceneManager

Metodi Principali

  1. LoadScene(int sceneBuildIndex) 
    - Carica una scena specificata dal suo indice di costruzione.
- **Overload:** `LoadScene(string sceneName)`
- **Overload:** `LoadScene(string sceneName, LoadSceneMode mode)`
- **Overload:** `LoadScene(int sceneBuildIndex, LoadSceneMode mode)`
    Esempio:
SceneManager.LoadScene(1); // Carica la scena con l'indice 1
SceneManager.LoadScene("MainScene"); // Carica la scena chiamata "MainScene"
SceneManager.LoadScene("MainScene", LoadSceneMode.Additive); // Carica la scena in modalità additiva
  1. LoadSceneAsync(int sceneBuildIndex)
  • Carica una scena in modo asincrono dal suo indice di costruzione.
  • Overload: LoadSceneAsync(string sceneName)
  • Overload: LoadSceneAsync(string sceneName, LoadSceneMode mode)
  • Overload: LoadSceneAsync(int sceneBuildIndex, LoadSceneMode mode)
    Esempio:
SceneManager.LoadSceneAsync("MainScene"); // Carica la scena in modo asincrono
  1. UnloadSceneAsync(int sceneBuildIndex)
  • Scarica una scena in modo asincrono dal suo indice di costruzione.
  • Overload: UnloadSceneAsync(string sceneName)
    Esempio:
SceneManager.UnloadSceneAsync("MainScene"); // Scarica la scena in modo asincrono
  1. GetActiveScene()
  • Restituisce la scena attualmente attiva.
    Esempio:
Scene activeScene = SceneManager.GetActiveScene();
  1. SetActiveScene(Scene scene)
  • Imposta una scena specificata come scena attiva.
    Esempio:
SceneManager.SetActiveScene(scene); // Imposta una scena come attiva
  1. GetSceneByName(string name)
  • Restituisce una scena caricata in memoria, identificata dal suo nome.
    Esempio:
Scene scene = SceneManager.GetSceneByName("MainScene");
  1. GetSceneByBuildIndex(int buildIndex)
  • Restituisce una scena caricata in memoria, identificata dal suo indice di costruzione.
    Esempio:
Scene scene = SceneManager.GetSceneByBuildIndex(1);
  1. GetSceneAt(int index)
  • Restituisce la scena caricata all'indice specificato nell'elenco delle scene caricate.
    Esempio:
Scene scene = SceneManager.GetSceneAt(0); // Ottiene la prima scena caricata
  1. sceneCount
  • Restituisce il numero totale di scene attualmente caricate.
    Esempio:
int count = SceneManager.sceneCount;
  1. sceneCountInBuildSettings
  • Restituisce il numero totale di scene elencate nelle impostazioni di costruzione del progetto.
    Esempio:
int count = SceneManager.sceneCountInBuildSettings;

  • sceneLoaded: Evento chiamato quando una scena viene caricata.

{
    // Azioni da eseguire quando una scena viene scaricata
}
  • activeSceneChanged: Evento chiamato quando la scena attiva cambia.
SceneManager.activeSceneChanged += OnActiveSceneChanged;

void OnActiveSceneChanged(Scene oldScene, Scene newScene)
{

Esempi di Utilizzo

Caricare una Scena in Modalità Single
using UnityEngine;
using UnityEngine.SceneManagement;

public class SceneLoader : MonoBehaviour
{
    void LoadNewScene()
    {
        SceneManager.LoadScene("GameScene");
    }
}
Caricare una Scena in Modalità Additive
using UnityEngine;
using UnityEngine.SceneManagement;

public class SceneLoader : MonoBehaviour
{
    void LoadAdditiveScene()
    {
        SceneManager.LoadScene("AdditionalScene", LoadSceneMode.Additive);
    }
}
Scaricare una Scena in Modo Asincrono
using UnityEngine;
using UnityEngine.SceneManagement;

public class SceneUnloader : MonoBehaviour
{
    void UnloadScene()
    {
        SceneManager.UnloadSceneAsync("OldScene");
    }
}

Quando Utilizzare SceneManager

  • Gestione delle Scene: Per caricare, scaricare e gestire le scene nel gioco, come passare da un livello all'altro o caricare scene additive per un'esperienza di gioco più complessa.
  • Ottimizzazione del Caricamento: Per caricare scene in modo asincrono e migliorare l'esperienza utente riducendo i tempi di caricamento percepiti.
  • Monitoraggio e Debug: Per gestire eventi di caricamento e scaricamento delle scene, permettendoti di eseguire operazioni specifiche durante il ciclo di vita di una scena.

Conclusione

La classe SceneManager è essenziale per la gestione delle scene in Unity. Ti permette di avere il controllo completo sul caricamento, lo scaricamento e la gestione delle scene, migliorando l'organizzazione del tuo gioco o applicazione e ottimizzando le prestazioni complessive.
