Physics classe

La classe Physics nel namespace UnityEngine è una classe statica che fornisce vari metodi e proprietà per la gestione e il controllo del motore fisico di Unity. Questo include il controllo delle collisioni, l'esecuzione di raycasting, l'interazione con i Rigidbody e molto altro. È una delle classi fondamentali per lavorare con la fisica all'interno di Unity.

Dettagli sulla Classe Physics

Namespace

  • Namespace: UnityEngine

Descrizione

La classe Physics offre un'interfaccia per l'interazione con il sistema di fisica di Unity. Essendo una classe statica, non è possibile istanziarla direttamente. Invece, puoi accedere alle sue funzionalità direttamente tramite il nome della classe, ad esempio Physics.Raycast().

Principali Funzionalità

1. Raycasting

  • Physics.Raycast(): Esegue un raycast, ovvero lancia un raggio virtuale dalla posizione iniziale in una direzione specificata per rilevare collisioni con collider presenti nella scena. Questo è utile per rilevare oggetti, gestire visibilità o determinare percorsi di navigazione.
if (Physics.Raycast(transform.position, transform.forward, out RaycastHit hit, 100))
{
    Debug.Log("Hit: " + hit.collider.name);
}
  • Physics.RaycastAll(): Esegue un raycast che rileva tutte le collisioni lungo il raggio e restituisce un array di tutti gli oggetti colpiti.
RaycastHit[] hits = Physics.RaycastAll(transform.position, transform.forward, 100);
foreach (RaycastHit hit in hits)
{
    Debug.Log("Hit: " + hit.collider.name);
}

2. Collision Detection

  • Physics.CheckSphere(): Verifica se ci sono collider all'interno di una sfera immaginaria di un determinato raggio e posizione. È utile per rilevare la presenza di oggetti in una determinata area.
bool isOverlapping = Physics.CheckSphere(transform.position, 5f);
  • Physics.OverlapSphere(): Restituisce un array di collider che si trovano all'interno di una sfera immaginaria di un determinato raggio e posizione.
Collider[] colliders = Physics.OverlapSphere(transform.position, 5f);
foreach (Collider collider in colliders)
{
    Debug.Log("Found object: " + collider.name);
}
  • Physics.CheckBox(): Verifica se ci sono collider all'interno di un volume a forma di scatola.
bool isOverlappingBox = Physics.CheckBox(transform.position, new Vector3(1, 1, 1));

3. Gestione dei Rigidbody

  • Physics.IgnoreCollision(): Imposta due collider in modo che ignorino le collisioni tra loro.
Physics.IgnoreCollision(collider1, collider2);
  • Physics.IgnoreLayerCollision(): Configura le collisioni tra due livelli di oggetti in modo che vengano ignorate.
Physics.IgnoreLayerCollision(8, 9);

4. Gestione delle Forze

  • Physics.gravity: Definisce o restituisce la gravità globale applicata a tutti i Rigidbody nella scena.
Physics.gravity = new Vector3(0, -9.81f, 0);
  • Physics.defaultContactOffset: Definisce o restituisce la distanza minima di contatto tra collider che innesca una collisione.
Physics.defaultContactOffset = 0.01f;

5. Miscellanea

  • Physics.GetIgnoreLayerCollision(): Verifica se le collisioni tra due livelli sono ignorate.
bool isIgnoring = Physics.GetIgnoreLayerCollision(8, 9);
  • Physics.Simulate(): Simula un passo fisico manualmente. Questo è utile se vuoi un maggiore controllo sul ciclo di simulazione della fisica.
Physics.Simulate(Time.deltaTime);
  • Physics.RaycastNonAlloc(): Esegue un raycast come Physics.Raycast(), ma senza allocare memoria per ogni chiamata. È più efficiente in termini di prestazioni.
RaycastHit[] hits = new RaycastHit[10];
int hitCount = Physics.RaycastNonAlloc(ray, hits);

Esempio di Utilizzo

Ecco un esempio di come utilizzare alcune delle funzionalità della classe Physics:

using UnityEngine;

public class PhysicsExample : MonoBehaviour
{
    void Update()
    {
        // Raycast dalla posizione dell'oggetto in avanti
        if (Physics.Raycast(transform.position, transform.forward, out RaycastHit hit, 100))
        {
            Debug.Log("Hit: " + hit.collider.name);
        }

        // Cambia la gravità globale
        Physics.gravity = new Vector3(0, -10f, 0);
    }
}

Conclusione

La classe Physics è una parte essenziale del motore fisico di Unity, fornendo una vasta gamma di metodi per controllare e gestire la fisica in un progetto Unity. È fondamentale per la gestione delle collisioni, delle forze e dell'interazione tra gli oggetti nel mondo di gioco.
