For Python

Il ciclo for in Python viene utilizzato per iterare su una sequenza (che può essere una lista, una tupla, un dizionario, un set o una stringa). A differenza di altri linguaggi di programmazione che usano un contatore per iterare su un intervallo di valori, in Python il ciclo for scorre direttamente sugli elementi della sequenza.

Sintassi del Ciclo for

for elemento in sequenza:
    # blocco di codice
  • elemento: è la variabile che prende successivamente il valore di ogni elemento della sequenza.
  • sequenza: è la collezione di elementi da iterare (lista, stringa, tupla, dizionario, ecc.).

Esempio con Lista

numeri = [1, 2, 3, 4, 5]
for numero in numeri:
    print(numero)

Output:

1
2
3
4
5

In questo esempio, la variabile numero prende successivamente il valore di ciascun elemento della lista numeri.

Esempio con Stringa

parola = "Python"
for lettera in parola:
    print(lettera)

Output:

P
y
t
h
o
n

Qui il ciclo for scorre ogni carattere della stringa parola.

Esempio con Tuple

tupla = (10, 20, 30, 40)
for valore in tupla:
    print(valore)

Output:

10
20
30
40

Anche le tuple possono essere iterate come le liste.

Esempio con Dizionario

Puoi usare il ciclo for per iterare sulle chiavi di un dizionario o sia su chiavi che valori.

Iterazione sulle Chiavi

dizionario = {'a': 1, 'b': 2, 'c': 3}
for chiave in dizionario:
    print(chiave)

Output:

a
b
c

Iterazione su Chiavi e Valori

for chiave, valore in dizionario.items():
    print(f"{chiave}: {valore}")

Output:

a: 1
b: 2
c: 3

In questo caso, il metodo .items() restituisce sia la chiave che il valore.

Ciclo for con range()

La funzione range() viene spesso usata per generare una sequenza di numeri e iterare su di essa.

Sintassi di range()

range(start, stop, step)
  • start: valore di partenza (incluso, predefinito è 0).
  • stop: valore finale (escluso).
  • step: incremento (predefinito è 1).

Esempio con range()

for i in range(5):
    print(i)

Output:

0
1
2
3
4

Il ciclo itera sui numeri da 0 a 4. range(5) genera una sequenza di numeri da 0 a 4.

Esempio con start e step

for i in range(2, 10, 2):
    print(i)

Output:

2
4
6
8

In questo caso, range(2, 10, 2) genera i numeri da 2 a 8 con un passo di 2.

Ciclo for Annidato

Puoi annidare cicli for per iterare su sequenze multiple o complesse.

Esempio

for i in range(3):
    for j in range(2):
        print(f"i = {i}, j = {j}")

Output:

i = 0, j = 0
i = 0, j = 1
i = 1, j = 0
i = 1, j = 1
i = 2, j = 0
i = 2, j = 1

Qui abbiamo due cicli annidati: il primo scorre i valori da 0 a 2, il secondo da 0 a 1 per ogni iterazione del primo ciclo.

break e continue nei Cicli for

break

L'istruzione break interrompe il ciclo for prima che la sequenza sia completata.

for i in range(5):
    if i == 3:
        break
    print(i)

Output:

0
1
2

Il ciclo si interrompe quando i raggiunge 3.

continue

L'istruzione continue salta l'iterazione corrente e passa direttamente alla successiva.

for i in range(5):
    if i == 2:
        continue
    print(i)

Output:

0
1
3
4

Quando i è 2, il ciclo salta quell'iterazione e continua con il valore successivo.

Ciclo for con else

Un ciclo for può avere un blocco else. Il blocco else viene eseguito quando il ciclo finisce normalmente, senza essere interrotto da break.

Esempio con else

for i in range(3):
    print(i)
else:
    print("Ciclo terminato normalmente")

Output:

0
1
2
Ciclo terminato normalmente

Se il ciclo viene interrotto con break, l'else non viene eseguito:

for i in range(3):
    if i == 1:
        break
    print(i)
else:
    print("Ciclo terminato normalmente")

Output:

0

In questo caso, il ciclo si interrompe a i == 1, quindi il blocco else non viene eseguito.

Conclusione

Il ciclo for in Python è uno strumento potente per iterare su sequenze. Può essere utilizzato con diverse collezioni come liste, tuple, dizionari e stringhe, e funziona bene con funzioni come range() per iterare su sequenze di numeri. Inoltre, con l'uso di break, continue, e else, il ciclo for offre un controllo molto fine sul flusso del programma.
