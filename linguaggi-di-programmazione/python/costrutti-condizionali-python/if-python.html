if Python

L'istruzione if permette di eseguire un blocco di codice se una condizione è vera.

Costrutto Condizionale if, elif, else in Python

1. if: la condizione di base

L'istruzione if verifica una condizione espressa come un'espressione booleana. Se la condizione è vera, esegue il blocco di codice associato. Se è falsa, salta il blocco.

Sintassi

if condizione:
    # blocco di codice se la condizione è vera

Esempio

x = 10

if x > 5:
    print("x è maggiore di 5")

Output:

x è maggiore di 5

In questo caso, l'istruzione print() viene eseguita perché la condizione x > 5 è vera.

2. else: la condizione di fallback

L'istruzione else viene utilizzata per eseguire un blocco di codice se la condizione if è falsa. È una "via di fuga" che esegue qualcosa nel caso in cui tutte le condizioni siano false.

Sintassi

if condizione:
    # blocco di codice se la condizione è vera
else:
    # blocco di codice se la condizione è falsa

Esempio

x = 3

if x > 5:
    print("x è maggiore di 5")
else:
    print("x è minore o uguale a 5")

Output:

x è minore o uguale a 5

3. elif: condizioni multiple

Se vuoi verificare più condizioni oltre alla prima, usi elif (abbreviazione di "else if"). Puoi avere quanti elif vuoi per gestire diverse condizioni.

Sintassi

if condizione1:
    # blocco di codice se condizione1 è vera
elif condizione2:
    # blocco di codice se condizione1 è falsa e condizione2 è vera
elif condizione3:
    # blocco di codice se condizione1 e condizione2 sono false e condizione3 è vera
else:
    # blocco di codice se tutte le condizioni precedenti sono false

Esempio

x = 10

if x > 10:
    print("x è maggiore di 10")
elif x == 10:
    print("x è uguale a 10")
else:
    print("x è minore di 10")

Output:

x è uguale a 10
  • Se x > 10 fosse stato vero, avrebbe eseguito il primo blocco.
  • Poiché x == 10 è vero, esegue il blocco di codice dell'elif.
  • Se nessuna delle condizioni fosse stata vera, avrebbe eseguito il blocco dell'else.

4. Annidamento di Condizioni

In Python, puoi annidare istruzioni if, elif, else all'interno di altre per gestire condizioni più complesse.

Sintassi

if condizione1:
    # blocco di codice se condizione1 è vera
    if condizione2:
        # blocco di codice se condizione2 è vera
    else:
        # blocco di codice se condizione2 è falsa
else:
    # blocco di codice se condizione1 è falsa

Esempio

x = 15

if x > 10:
    print("x è maggiore di 10")
    
    if x < 20:
        print("x è minore di 20")
    else:
        print("x è maggiore o uguale a 20")
else:
    print("x è minore o uguale a 10")

Output:

x è maggiore di 10
x è minore di 20

In questo esempio:

  • La prima condizione x > 10 è vera, quindi il primo blocco viene eseguito.
  • All'interno di questo blocco, c'è un'altra condizione: x < 20, che è anch'essa vera, quindi viene eseguito il secondo blocco.

Puoi annidare quanti if vuoi, ma devi stare attento a mantenere leggibile il codice.

5. Condizioni Multiple con Operatori Logici (and, or, not)

Puoi combinare più condizioni all'interno di una singola istruzione if usando gli operatori logici come and, or, e not.

Esempio con and

x = 7

if x > 5 and x < 10:
    print("x è compreso tra 5 e 10")

Output:

x è compreso tra 5 e 10

Esempio con or

x = 3

if x < 5 or x > 10:
    print("x è minore di 5 o maggiore di 10")

Output:

x è minore di 5 o maggiore di 10

Esempio con not

x = False

if not x:
    print("x è False")

Output:

x è False

6. Condizioni su una Sola Riga (Operatore Ternario)

Python supporta un'istruzione condizionale su una sola riga, chiamata operatore ternario, che ti permette di assegnare un valore in base a una condizione.

Sintassi

valore = valore_true if condizione else valore_false

Esempio

x = 10
messaggio = "x è maggiore di 5" if x > 5 else "x non è maggiore di 5"
print(messaggio)

Output:

x è maggiore di 5

Conclusione

Il costrutto condizionale if, elif, else in Python è molto potente e flessibile, permettendo di gestire una vasta gamma di condizioni e situazioni logiche. Grazie all'uso di operatori logici e annidamenti, puoi costruire condizioni complesse e gestire casi diversi in modo ordinato. Python rende queste operazioni molto intuitive grazie alla sua sintassi semplice e leggibile.
