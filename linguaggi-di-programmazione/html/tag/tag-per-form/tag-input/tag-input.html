Tag input

Il tag <input> in HTML viene utilizzato per creare vari tipi di controlli interattivi in un modulo web, come campi di testo, pulsanti di invio, caselle di controllo, radio button, e altro ancora. È uno degli elementi più versatili e fondamentali per raccogliere dati dagli utenti.

Sintassi del Tag <input>

<input type="tipo" name="nome" value="valore">

Tipi di Input

  • Type text: Campo di testo a riga singola.
  • Type password: Campo di testo in cui i caratteri inseriti sono mascherati (non visibili).
  • Type email: Campo di testo che valida l'input come un indirizzo email.
  • Type url: Campo di testo che valida l'input come un URL.
  • Type tel: Campo di testo per inserire numeri di telefono.
  • Type number: Campo per inserire numeri, con la possibilità di definire un intervallo (min, max) e un passo (step).
  • Type range: Un controllo a scorrimento per selezionare un valore numerico all'interno di un intervallo.
  • Type date: Campo per selezionare una data.
  • Type time: Campo per selezionare un orario.
  • Type datetime-local: Campo per selezionare una data e un'ora (senza fuso orario).
  • Type month: Campo per selezionare un mese e un anno.
  • Type week: Campo per selezionare una settimana e un anno.
  • Type color: Campo per selezionare un colore tramite una tavolozza.
  • Type checkbox: Casella di controllo per selezionare o deselezionare un'opzione.
  • Type radio: Pulsante di opzione per selezionare una scelta da un gruppo di opzioni.
  • Type file: Campo per selezionare e caricare un file.
  • Type hidden: Campo nascosto che non è visibile all'utente, utilizzato per trasmettere dati.
  • Type submit: Pulsante per inviare il modulo.
  • Type reset : Pulsante per resettare (azzerare) tutti i campi del modulo.
  • Type button : Pulsante generico che può essere associato a un'azione tramite JavaScript.
  • Type image : Pulsante di invio del modulo che utilizza un'immagine.
  • Type search : Campo di testo per l'inserimento di query di ricerca, con alcune funzionalità aggiuntive come la cancellazione rapida del testo.

Questa lista copre i principali tipi di input che puoi usare con il tag <input> in HTML5. Ognuno di questi tipi ha i propri attributi e caratteristiche specifiche, e può essere utilizzato per costruire moduli ricchi di funzionalità e user-friendly.
