Type text

Il tag <input> con type="text" è utilizzato per creare un campo di input di testo a riga singola. Questo è il tipo di input più comune e versatile, adatto per raccogliere qualsiasi tipo di dati di testo dagli utenti.

Sintassi di Base

<input type="text" name="name">

Attributi Comuni

  • type="text": Specifica che l'input è di tipo testo.
  • name: Definisce il nome del campo, utilizzato per identificare il dato quando inviato al server.
  • value: Imposta il valore predefinito del campo.
  • placeholder: Testo che appare all'interno del campo quando è vuoto e scompare quando l'utente inizia a digitare.
  • maxlength: Imposta il numero massimo di caratteri che possono essere inseriti.
  • required: Indica che il campo è obbligatorio e deve essere compilato prima di inviare il modulo.
  • readonly: Rende il campo di sola lettura; l'utente può vedere il testo ma non può modificarlo.
  • disabled: Disabilita il campo, rendendolo non modificabile e non invia i suoi dati.
  • size: Imposta la larghezza del campo di input in caratteri (questo attributo è stato in gran parte sostituito dall'uso di CSS per la dimensione).
  • autofocus: Specifica che l'input dovrebbe automaticamente ricevere il focus quando la pagina viene caricata.
  • pattern: Definisce un'espressione regolare che l'input deve rispettare per essere valido.
  • autocomplete: Suggerisce al browser se deve completare automaticamente il valore in base ai dati precedentemente inseriti.

Esempi di Utilizzo

  1. Campo di Testo Standard
<label for="username">Nome utente:</label>
<input type="text" id="username" name="username">
  1. Campo di Testo con Placeholder
<input type="text" name="fullname" placeholder="Inserisci il tuo nome completo">
  1. Campo di Testo con Limite di Caratteri
<input type="text" name="zipcode" maxlength="5" placeholder="CAP (max 5 caratteri)">
  1. Campo di Testo di Sola Lettura
<input type="text" name="readonlyfield" value="Non puoi modificarmi" readonly>
  1. Campo di Testo con Pattern di Validazione
<input type="text" name="phone" placeholder="000-000-0000" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" title="Il formato deve essere 000-000-0000">
  1. Campo di Testo con Valore Predefinito
<input type="text" name="city" value="Roma">

Utilizzo Avanzato

Puoi utilizzare CSS per migliorare l'aspetto del campo di testo e JavaScript per aggiungere funzionalità avanzate come la validazione dinamica o la gestione degli eventi.

<style>
  input[type="text"] {
    padding: 10px;
    font-size: 16px;
    border-radius: 5px;
    border: 1px solid #ccc;
  }
</style>

<input type="text" name="address" placeholder="Inserisci il tuo indirizzo">

Conclusione

Il campo di input di tipo testo (<input type="text">) è un elemento essenziale nei form HTML, utile per raccogliere vari tipi di dati in formato testo. Può essere personalizzato e adattato a diverse esigenze tramite l'uso di attributi e stili CSS.
