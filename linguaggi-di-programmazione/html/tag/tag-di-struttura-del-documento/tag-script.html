tag script

Il tag <script> in HTML è utilizzato per includere e eseguire script JavaScript all'interno di una pagina web. Gli script JavaScript possono essere utilizzati per aggiungere interattività, manipolare il DOM (Document Object Model), gestire eventi e molto altro.

Caratteristiche del Tag <script>:

  • Sintassi:
<script>
  // JavaScript code here
</script>
  • Posizione: Il tag <script> può essere posizionato sia all'interno del tag <head> che alla fine del tag <body>. La posizione può influenzare il momento in cui lo script viene eseguito e come interagisce con il contenuto della pagina.
  • Autoconclusivo: Non è autoconclusivo; deve essere chiuso con </script>.

Attributi Comuni del Tag <script>:

  1. src:

    • Descrizione: Specifica il percorso del file JavaScript esterno da includere. Quando è utilizzato, il contenuto tra i tag di apertura e chiusura <script> viene ignorato.
    • Esempio:
<script src="script.js"></script>
  1. type:
  • Descrizione: Specifica il tipo di script. Per i file JavaScript, il valore predefinito è text/javascript e in HTML5 è facoltativo.
  • Esempio (obsoleto in HTML5):
<script type="text/javascript" src="script.js"></script>
  1. async:
  • Descrizione: Indica al browser di eseguire lo script in modo asincrono. Lo script sarà scaricato in parallelo con il resto della pagina e sarà eseguito non appena il download è completato.
  • Esempio:
<script src="script.js" async></script>
  1. defer:
  • Descrizione: Indica al browser di eseguire lo script dopo che il documento è stato completamente caricato. Gli script con l'attributo defer sono eseguiti nell'ordine in cui sono definiti.
  • Esempio:
<script src="script.js" defer></script>
  1. integrity e crossorigin:
  • Descrizione: Utilizzati per la sicurezza e l'integrità dei file JavaScript esterni. Questi attributi sono più avanzati e sono utilizzati per garantire che il file non sia stato manomesso e per gestire le politiche di cross-origin.
  • Esempio:
<script src="script.js" integrity="sha384-..." crossorigin="anonymous"></script>

Esempio di Utilizzo:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pagina con Script</title>
    <!-- Inclusione di uno script esterno con l'attributo defer -->
    <script src="script.js" defer></script>
</head>
<body>
    <h1>Benvenuti alla Pagina con Script</h1>
    <p>Clicca il pulsante per vedere un messaggio.</p>
    <button onclick="showMessage()">Mostra Messaggio</button>

    <!-- Script interno -->
    <script>
        function showMessage() {
            alert('Ciao! Questo è un messaggio da un script interno.');
        }
    </script>
</body>
</html>

Considerazioni sull'Utilizzo del Tag <script>:

  • Posizionamento:
    • Nel <head>: Gli script posizionati nel <head> sono caricati e potenzialmente eseguiti prima che il contenuto della pagina sia completamente caricato. Questo può rallentare il rendering della pagina. L'uso di async e defer può mitigare questo problema.
    • Alla Fine del <body>: Posizionare gli script alla fine del <body> è una pratica comune per garantire che il contenuto della pagina sia completamente caricato prima che gli script vengano eseguiti. Questo evita problemi di accesso agli elementi del DOM prima che siano disponibili.
  • Performance: Gli attributi async e defer sono utili per migliorare le prestazioni di caricamento della pagina. async è usato quando l'ordine di esecuzione non è importante, mentre defer è usato per preservare l'ordine degli script.

In sintesi, il tag <script> è essenziale per integrare JavaScript nelle pagine web, permettendo di aggiungere funzionalità dinamiche e interattive. L'uso corretto degli attributi e la posizione del tag possono migliorare le prestazioni e l'efficacia degli script.
