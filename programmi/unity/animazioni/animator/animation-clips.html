Animation Clips

In Unity, un Animation Clip è un asset che definisce una sequenza di animazioni per un GameObject. Gli Animation Clips possono rappresentare qualsiasi cosa, da semplici movimenti di un personaggio a animazioni più complesse come cambi di forma o effetti speciali.

Componenti degli Animation Clips

  1. Keyframes:

    • I keyframes sono punti nel tempo che definiscono lo stato di un'animazione. Ogni keyframe memorizza la posizione, rotazione e scala di un GameObject in un momento specifico. Unity utilizza questi keyframes per interpolare i dati tra di essi e creare una transizione fluida.

  2. Curves:

    • Le curve definiscono l'andamento dei cambiamenti delle proprietà del GameObject nel tempo. Ogni proprietà animabile (come la posizione o la scala) ha una curva associata che determina come il valore cambia durante l'animazione.

  3. Events:

    • Gli eventi di animazione possono essere aggiunti a un Animation Clip per eseguire azioni specifiche a determinati punti nel tempo dell'animazione, come suoni o modifiche di stato.

Creare e Configurare Animation Clips

1. Creare un Animation Clip

  • Utilizzare l'Animation Window:
    • Apri l'Animation Window tramite Window > Animation > Animation.
    • Seleziona il GameObject che vuoi animare.
    • Clicca su Create nella finestra dell'Animation per creare un nuovo Animation Clip. Ti verrà chiesto di salvare l'Animation Clip nella tua cartella Assets.
  • Creare un Clip vuoto:
    • Puoi anche creare un Animation Clip vuoto direttamente nel pannello Project con Assets > Create > Animation. Poi lo puoi assegnare a un GameObject tramite il componente Animator.

2. Modificare un Animation Clip

  • Utilizzare l'Animation Window:
    • Con il GameObject selezionato e l'Animation Window aperta, puoi iniziare a aggiungere keyframes alle proprietà del GameObject.
    • Cambia il valore di una proprietà del GameObject e aggiungi un keyframe cliccando sul pulsante Add Keyframe (icona a forma di rombo).
  • Modificare Curves:
    • Nella finestra dell'Animation, puoi passare alla vista Curves per vedere e modificare le curve di animazione associate al tuo Animation Clip. Le curve mostrano come cambiano le proprietà nel tempo.
  • Aggiungere Eventi:
    • Vai alla vista Events dell'Animation Window per aggiungere eventi specifici a determinati punti temporali dell'animazione. Gli eventi possono essere utilizzati per chiamare metodi specifici su componenti o script.

3. Assegnare un Animation Clip a un Animator Controller

  • Aprire l'Animator Controller:
    • Crea o apri un Animator Controller tramite il pannello Project.
  • Aggiungere un Animation Clip:
    • Trascina il tuo Animation Clip dalla cartella Assets nel grafico dell'Animator Controller. Questo creerà uno stato di animazione associato al clip.
  • Configurare Transizioni:
    • Collega gli stati di animazione con transizioni e imposta le condizioni basate sui parametri per determinare quando passare da un'animazione all'altra.

Esempio di Utilizzo di Animation Clips

Supponiamo di avere un personaggio con un'animazione di camminata e una di salto.

1. Creare le Animazioni

  • Creare il Clip di Camminata:
    • Seleziona il personaggio nella scena, apri l'Animation Window e crea un nuovo Animation Clip chiamato "Walk".
    • Aggiungi keyframes per animare la camminata del personaggio.
  • Creare il Clip di Salto:
    • Crea un altro Animation Clip chiamato "Jump" e aggiungi keyframes per l'animazione di salto.

2. Configurare l'Animator Controller

  • Aggiungere gli Stati:
    • Trascina "Walk" e "Jump" nel grafico dell'Animator Controller per creare stati di animazione.
  • Impostare Transizioni:
    • Crea transizioni tra gli stati, ad esempio, da "Idle" a "Walk" quando un parametro "Speed" è maggiore di 0.1 e da "Walk" a "Jump" quando un parametro "IsJumping" è vero.

3. Controllare le Animazioni tramite Script

using UnityEngine;

public class PlayerController : MonoBehaviour
{
    public Animator animator;

    void Update()
    {
        float move = Input.GetAxis("Vertical");
        animator.SetFloat("Speed", move);

        if (Input.GetKeyDown(KeyCode.Space))
        {
            animator.SetBool("IsJumping", true);
        }
        else if (Input.GetKeyUp(KeyCode.Space))
        {
            animator.SetBool("IsJumping", false);
        }
    }
}

In questo script, i parametri "Speed" e "IsJumping" vengono utilizzati per controllare quale animazione deve essere riprodotta.

Considerazioni Avanzate

  • Looping: Gli Animation Clips possono essere configurati per loopare, utile per animazioni cicliche come camminare o correre.
  • Blend Trees: Utilizzati per gestire animazioni più complesse che dipendono da più parametri, come mescolare camminata e corsa a diverse velocità.
  • Root Motion: Se abilitato, il movimento del GameObject può essere controllato dall'animazione stessa, piuttosto che da script o fisica.

Conclusione

Gli Animation Clips sono essenziali per animare i tuoi GameObject in Unity, fornendo la base per creare sequenze di animazioni fluide e reattive. Configurare e gestire correttamente gli Animation Clips ti permette di arricchire l'esperienza di gioco e di migliorare l'interazione con il tuo personaggio o oggetto animato.
