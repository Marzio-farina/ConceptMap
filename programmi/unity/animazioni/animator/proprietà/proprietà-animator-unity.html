Proprietà Animator Unity

Proprietà dell'Animator

  1. runtimeAnimatorController Unity

    • Ottiene o imposta il RuntimeAnimatorController assegnato all'Animator, che definisce le animazioni e le transizioni.
    • Esempio
    RuntimeAnimatorController controller = animator.runtimeAnimatorController;

  1. hasBoundPlayables Unity

    • Indica se l'Animator ha degli "Playable" associati.
    • Esempio
	bool boundPlayables = animator.hasBoundPlayables;

  1. avatar Unity

    • Ottiene o imposta l'Avatar utilizzato dall'Animator per gestire le animazioni basate sull'osso.
    • Esempio:
	Avatar avatar = animator.avatar;

  1. applyRootMotion Unity

    • Indica se il movimento dell'animazione deve essere applicato al GameObject.
    • Esempio
	animator.applyRootMotion = true;

  1. enabled Unity

    • Abilita o disabilita il componente Animator. Quando è disabilitato, l'Animator non riproduce animazioni.
    • Esempio
	animator.enabled = false;

  1. speed Unity

    • Imposta la velocità con cui le animazioni vengono riprodotte. Un valore di 1 significa la velocità normale, mentre valori superiori aumentano la velocità e valori inferiori la riducono.
    • Esempio
	animator.speed = 1.5f; // Riproduce le animazioni al 150% della velocità normale

  1. keepAnimatorControllerStateOnDisable Unity

    • Indica se mantenere lo stato dell'AnimatorController quando il componente Animator viene disabilitato.
    • Esempio:
	animator.keepAnimatorControllerStateOnDisable = true;

  1. updateMode Unity

    • Specifica il modo in cui l'Animator viene aggiornato. Può essere impostato su AnimatorUpdateMode.Normal, AnimatorUpdateMode.AnimatePhysics, o AnimatorUpdateMode.UnscaledTime.
    • Esempio:
	animator.updateMode = AnimatorUpdateMode.AnimatePhysics;

  1. hasTransformHierarchy Unity

    • Indica se l'Animator ha una gerarchia di Transform associata. Questa proprietà è utile per determinare se l'Animator è completamente configurato con un avatar e le sue ossa.
    • Esempio:
	bool hasTransforms = animator.hasTransformHierarchy;

  1. layerCount Unity

    • Ottiene il numero di layer definiti nell'AnimatorController. I layer sono utilizzati per organizzare e combinare diverse animazioni.
    • Esempio:
	int numberOfLayers = animator.layerCount;
  1. rootPosition Unity
  • Ottiene o imposta la posizione radice del GameObject associato all'Animator.
  • Esempio:
	Vector3 rootPos = animator.rootPosition;
    animator.rootPosition = new Vector3(0, 0, 0);
  1. rootRotation Unity
  • Ottiene o imposta la rotazione radice del GameObject associato all'Animator.
  • Esempio:
	Quaternion rootRot = animator.rootRotation;
    animator.rootRotation = Quaternion.identity;
  1. bodyPosition Unity
  • Ottiene o imposta la posizione del corpo del rig umanoide.
  • Esempio:
	Vector3 bodyPos = animator.bodyPosition;
    animator.bodyPosition = new Vector3(0, 1, 0);
  1. bodyRotation Unity
  • Ottiene o imposta la rotazione del corpo del rig umanoide.
  • Esempio:
	Quaternion bodyRot = animator.bodyRotation;
    animator.bodyRotation = Quaternion.Euler(0, 90, 0);
  1. cullingMode Unity
  • Ottiene o imposta la modalità di culling dell'Animator. Può essere impostata su AnimatorCullingMode.AlwaysAnimate, AnimatorCullingMode.CullUpdateTransforms, o AnimatorCullingMode.CullCompletely.
  • Esempio:
	animator.cullingMode = AnimatorCullingMode.CullCompletely;
  1. playbackTime Unity
  • Ottiene o imposta il tempo di riproduzione dell'Animator. Può essere usato per sincronizzare o controllare la riproduzione delle animazioni.
  • Esempio:
    animator.playbackTime = 2.0f;
  1. recorderStartTime Unity
  • Ottiene o imposta il tempo di inizio del registratore dell'Animator.
  • Esempio:
	animator.recorderStartTime = 0.5f;
  1. recorderStopTime Unity
  • Ottiene o imposta il tempo di fine del registratore dell'Animator.
  • Esempio:
	animator.recorderStopTime = 3.0f;
  1. stabilizeFeet Unity
  • Indica se i piedi del personaggio devono essere stabilizzati durante la riproduzione delle animazioni.
  • Esempio:
    animator.stabilizeFeet = true;
  1. fireEvents Unity
  • Indica se l'Animator deve inviare eventi durante la riproduzione delle animazioni.
  • Esempio:
	animator.fireEvents = true;
