4xx – Client Error (Errore del Client)

I codici della serie 4xx indicano che c'è stato un errore nella richiesta inviata dal client.

  • 400 Bad Request: La richiesta è malformata o non comprensibile dal server.
  • 401 Unauthorized: Autenticazione richiesta per accedere alla risorsa.
  • 402 Payment Required: Codice riservato per usi futuri, solitamente legato a pagamenti.
  • 403 Forbidden: Accesso alla risorsa negato, anche se l'autenticazione è stata eseguita.
  • 404 Not Found: La risorsa richiesta non è stata trovata.
  • 405 Method Not Allowed: Il metodo HTTP usato nella richiesta non è consentito per quella risorsa.
  • 406 Not Acceptable: La risorsa non è disponibile in un formato accettabile per il client.
  • 407 Proxy Authentication Required: Autenticazione richiesta dal proxy.
  • 408 Request Timeout: Il server ha impiegato troppo tempo per ricevere la richiesta.
  • 409 Conflict: La richiesta non può essere completata a causa di un conflitto con lo stato corrente della risorsa.
  • 410 Gone: La risorsa non è più disponibile e non sarà ripristinata.
  • 411 Length Required: Il server richiede che la richiesta specifichi la lunghezza del contenuto.
  • 412 Precondition Failed: Le condizioni specificate nel campo dell'intestazione della richiesta non sono state soddisfatte.
  • 413 Payload Too Large: Il payload della richiesta è troppo grande per essere elaborato dal server.
  • 414 URI Too Long: L'URL richiesto è troppo lungo da essere processato dal server.
  • 415 Unsupported Media Type: Il formato del payload della richiesta non è supportato dal server.
  • 416 Range Not Satisfiable: La richiesta specifica una parte della risorsa che non può essere fornita.
  • 417 Expectation Failed: Il server non può soddisfare i requisiti specificati nell'intestazione Expect.
  • 418 I'm a teapot: Un codice umoristico definito nel protocollo Hyper Text Coffee Pot Control.
  • 421 Misdirected Request: La richiesta è stata indirizzata a un server che non è in grado di produrre una risposta.
  • 422 Unprocessable Entity: La richiesta era ben formata ma non può essere elaborata a causa di errori semantici.
  • 423 Locked: La risorsa è bloccata.
  • 424 Failed Dependency: La richiesta fallisce perché una richiesta precedente da cui dipende non è riuscita.
  • 425 Too Early: Il server non è disposto a elaborare la richiesta perché potrebbe essere ripetuta in futuro.
  • 426 Upgrade Required: Il client deve aggiornare il protocollo per procedere.
  • 428 Precondition Required: Il server richiede che la richiesta includa determinate precondizioni.
  • 429 Too Many Requests: Il client ha inviato troppe richieste in un breve periodo di tempo (rate limiting).
  • 431 Request Header Fields Too Large: I campi di intestazione della richiesta sono troppo grandi.
  • 451 Unavailable For Legal Reasons: La risorsa è stata bloccata per ragioni legali (ad esempio, richieste di censura).
